1. The ôBoxö Metaphor: Variables

Core concept

Think of a variable as a labeled box. You put a value inside the box so you can remember it and use it later.

python 
# To create a variable, just give it a name and a value
user_name = "Alex"
user_age = 25

Key Rules for Names:

  • Use underscores _ instead of spaces.
  • Start with a letter (not a number).
  • Case matters: Age and age are two different boxes.

3. Lists: The Organized Shelf

Collection

Sometimes one box isn't enough. A List lets you keep multiple items in a specific order.

  • Use square brackets to define a list: []
  • Order matters: items keep the position you give them.
  • Indexing starts at 0: the first item is [0].
python 
# Lists are defined with square brackets []
coding_languages = ["Python", "JavaScript", "C++"]

# You can access items by their position (starting at 0!)
print(coding_languages[0])  # This outputs: Python

4. Making Decisions: If Statements

Control flow

Logic is what makes a program "smart." We use if statements to check a condition. Indentation (the space at the start of the line) is how Python knows which code belongs inside the if block.

  • One condition starts with if.
  • Extra checks use elif (else-if).
  • Fallback uses else when nothing else matches.
python 
temperature = 22

if temperature > 30:
    print("It's a hot day!")
elif temperature > 15:
    print("It's a nice day.")
else:
    print("Brrr, it's cold!")

Note: The colon : and consistent indentation are requiredùotherwise Python will raise an error.
